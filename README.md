# Novapex ERP
A comprehensive ERP system designed for streamlined business operations.

## Key Features
* **Role-Based Access:** Specific interfaces for HR, Managers, and Employees.
* **Employee Management:** Track profiles, payroll, and departments.
* **Attendance System:** Leave requests and time-tracking modules.
* **Performance:** Feedback, goals, and review systems.
